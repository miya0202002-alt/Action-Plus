# Traveling - 旅行×ファッション プラットフォーム

旅行計画における「ファッション」という文脈をDX（デジタルトランスフォーメーション）し、旅行準備を「作業」から「エンターテインメント」へと昇華させるプロジェクトです。

## 📋 プロジェクト概要

本プロジェクトは、旅行者の「装い」に関する不安をゼロにし、旅先での自己肯定感を最大化することを目的としたプラットフォームです。

### 主な機能

- **バーチャル試着体験**: アバターを通じて旅行先での服装を事前に確認
- **スマートクローゼット管理**: 手持ちの服をデジタル管理
- **コーディネートプランナー**: 旅行日程に合わせた日別コーディネート作成
- **リアルタイム気象データ連携**: 現地の天候に基づいた服装提案
- **OOTDログ機能**: 実際の旅行での服装記録

## 🎯 ターゲットユーザー

- 20代〜40代のファッションに関心がある層
- 年に2回以上の旅行を行う人
- SNSでの発信や記録を好む人

## 🚀 セットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/your-username/traveling.git
cd traveling
```

2. `index.html` をブラウザで開く
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```
